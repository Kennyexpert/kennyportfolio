SceneOne Video ChatApp

This Project was developed for the smooth communication between the staffs, crews and everyone working with sceneonepd.

The App allows you to communicate, hold meetings, chat with colleagues, etc.via video which allows the host and the attendees to see one another virtually.

Some of the features include: Host having the ability to mute both the audio and videos of the attendees.

The Attendee has the ability to mute his/her self audio, video or both.

Also, both parties (Host and attendee) can chat while the video meeting is on going.

Technologies and Language Used:
  • - Application Program Interface(API)
  • - VS code
  • - Node.js
  • - HTML
  • - CSS
  • - JavaScript
